## Usage

```shell
1.  git clone https://github.com/haihongren/weaver-template
2.  cd weaver-template
3.  ./gradlew createModule  
    ...Follow the prompt
    ...Type in <your package name>
    ...Type in <your module name>

4.  ./gradlew <your module name>:idea
5. import weaver-template project to to IntelliJ

```
